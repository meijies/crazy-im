# 压缩算法
为文本以及不同类型的文件选择合适的压缩算法，基于CPU，压缩率，压缩速度三个方面考虑。其次，对于图片和音频等媒体文件，还要区分有损压缩和
无损压缩。就IM系统来说，首要考虑的是压缩率，因为解压缩和压缩这个过程实在客户端进行的。
